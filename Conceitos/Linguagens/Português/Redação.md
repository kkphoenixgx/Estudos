# Redação
#Portugês 

Uma redação deve conter partes principais que devem ser elaboradas

1. Introdução
  * Tema
  * Tese
  * Gancho para o 
2. Desenvolvimento
	* 
3. Conclusão

- Conclusão no caso do enem = Proposta de intervenção (PI)
	- Agente -> Quem
	- Ação -> o que fazer
	-  Modo -> Como vai fazer 
	- Efeito -> Para que
	* Detalhamento -> Detalhar
